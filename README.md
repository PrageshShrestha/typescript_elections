# Nepal Election Data Visualizer

A full-stack application for the analysis and visualization of Nepal's election data.

## System Architecture

The project consists of a decoupled frontend and backend:
- **Frontend**: Next.js / React application.
- **Backend**: FastAPI server utilizing Pandas for data transformation and Devanagari numeral processing.



## Installation and Setup

Running this project requires two active terminal sessions.

### Prerequisites
- Python 3.10+
- Node.js 18+
- npm or yarn
